# Machine Payments (MPP)

Build machine payment flows that use Accounts SDK signing and Tempo payment primitives.

## Walkthrough

::::steps

### Install the SDK

```bash
pnpm add accounts mppx viem
```

### Configure the Provider

Choose the account adapter that signs machine payment requests.

### Connect the account

Load the account or access key used by the machine actor.

### Submit the request

Create and submit the MPP payment request.

### Verify the result

Confirm the payment status and persist the settlement result.
