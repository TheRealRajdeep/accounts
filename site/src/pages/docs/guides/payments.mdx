# Payments

Send a stablecoin payment from the connected account and keep the payment flow portable across adapters.

## Demo

Target workspace: `examples/payments/`.

:::warning[Stub]
This guide needs a dedicated runnable payment example. The current examples contain payment-like calls, but the final guide should choose the exact token, faucet behavior, and success state before publishing.
:::

## Walkthrough

::::steps

### Install the SDK

```bash
pnpm add accounts wagmi viem
```

### Configure the Provider

Use the same Wagmi configuration from [Connect Accounts](/docs/guides/connect-accounts).

### Connect the user's account

Connect through Tempo Wallet, WebAuthn, or a custom adapter.

### Submit the payment

Use Tempo actions or hooks to submit the transfer.

```tsx
import { parseUnits } from 'viem'
import { Actions } from 'viem/tempo'
import { useSendTransactionSync } from 'wagmi'

const pathUsd = '0x20c0000000000000000000000000000000000000'

function Pay() {
  const { sendTransactionSync } = useSendTransactionSync()

  return (
    <button
      onClick={() =>
        sendTransactionSync({
          calls: [
            Actions.token.transfer.call({
              token: pathUsd,
              to: '0x0000000000000000000000000000000000000001',
              amount: parseUnits('1', 6),
            }),
          ],
        })
      }
    >
      Pay
    </button>
  )
}
```

### Verify the result

Show the returned receipt and refresh the account balance.
