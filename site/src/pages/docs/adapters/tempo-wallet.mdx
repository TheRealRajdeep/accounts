# Tempo Wallet Adapter

Tempo Wallet is the hosted adapter for apps that want embedded account creation and signing without owning auth, passkeys, or wallet infrastructure.

Use it when you want:

- A universal account that can move across apps.
- Hosted approval surfaces through iframe or popup dialogs.
- Tempo-managed passkey and signing infrastructure.
- A fast path for app teams that do not need to build a wallet.

## Usage

:::code-group

```tsx twoslash [Web (Wagmi)]
import { createConfig, http } from 'wagmi'
import { tempo } from 'wagmi/chains'
import { tempoWallet } from 'wagmi/connectors'

export const config = createConfig({
  chains: [tempo],
  connectors: [tempoWallet()],
  transports: {
    [tempo.id]: http(),
  },
})
```

```ts twoslash [Web (Vanilla)]
import { Provider, tempoWallet } from 'accounts'

const provider = Provider.create({
  adapter: tempoWallet(),
})
```

```ts twoslash [CLI]
import { Provider } from 'accounts/cli'

const provider = Provider.create()
```

:::
